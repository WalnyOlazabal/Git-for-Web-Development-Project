Questions: 
1. What is Semantic HTML? 
-It is the correct way to use html to reinforce the meaning of content on a web page.
2. What is HTML used for? 
-HTML is used to serve as the foundation for a web page, you can use it to add text,images, and links.
3. What is an attribute and where do we put it? 
-Attributes are things that define characteristics of elements in HTML, they are written inside the opening tags.
4. What is the h1 tag used for? How many times should I use it on a page?
-The h1 tag is used to mark the title of the page. It should only be used once per page.
5. Name two tags that have required attributes
-<img> and <a>.
6. What do we put in the head of our HTML document? 
-Data that is not displayed on the page, such as links to CSS.
7. What is an id?
-An id is an attribute that marks an element, it can only be used in 1 element.
8. What elements can I add an id to?
-Any element, but only once. 
9. How many times can I use the same id on a page? 
-Only once.
10. What is a class?
-It is an attribute used to mark an element, for example, for styling purposes, similar to an id
11. What elements can I add a class to? 
-Any element.
12. How many times can I use the same class on a page? 
-There is no limit.
13. How do I get my link to open in a new tab?
-By adding the target attribute.
14. What is the alt attribute in the image tag used for?
-To have something to display in case the image does not display. 
15. How do I reference an id?
-With a # symbol
16. What is the difference between a section and a div
-Section means the content is grouped and should appear as an entry in an outline of a page, 
whereas dive carries no meaning and it is just a generic text container.
17. What is CSS used for? 
-CSS is used for styling pages.

19. What is the difference between a class and an id? - Give me an example of when I might use each one
-Ypu use an id for a specific element you need to style in a very specific way, 
and a class when you are styling multiple elements in the same way.  
For example, you can use an id to stle the h1, but a class to style sections of a page.
